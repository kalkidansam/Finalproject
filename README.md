# Addis Ababa Living

A simple, responsive real estate website showcasing modern housing options in Addis Ababa.

## Features
- Home page with property listings
- Study planner page (LocalStorage)
- Contact form with basic validation
- Responsive layout (desktop & mobile)

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Run
Open index.html in a web browser.
